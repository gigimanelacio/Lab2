A tabela VALIDACOES é o coração do sistema. Cada picagem de pulseira regista quem entrou, por onde, e quando. As Foreign Keys garantem integridade referencial — não existe validação sem bilhete válido e sem porta registada.

VALIDACOES.id_bilhete → BILHETES.id

Cada validação pertence a um bilhete. Impede entradas sem bilhete registado.

VALIDACOES.id_porta → PORTAS.id

Cada validação está associada a uma porta física do recinto.

BILHETES.id_cliente → CLIENTES.id

Liga o bilhete ao proprietário. Essencial para processar reembolsos.

BLACKLIST RFID

Como garantir que uma pulseira reembolsada não entra offline?

1
Cliente pede reembolso online → estado muda para 'Reembolsado' na tabela BILHETES
2
RFID é adicionado à blacklist local dos leitores via sincronização periódica (a cada 60s ou push via websocket)
3
Leitor offline verifica o RFID primeiro na blacklist local antes de tentar qualquer rede
4
Se RFID está na blacklist → ACESSO NEGADO instantaneamente, sem precisar de Wi-Fi
5
Ao recuperar conexão, leitor sincroniza tentativas de fraude para auditoria na BD central

Análise Crítica

REFLEXÃO

01
// O que funcionou
Normalização salva o dia
Ter o artista numa tabela separada significa que bastou um UPDATE em concertos para propagar o cancelamento do Kanye a todo o sistema. Se o nome estivesse desnormalizado em cada bilhete, seriam 70.000 updates.
02
// O que funcionou
INDEX no RFID é crítico
Sem o índice na coluna rfid, cada picagem seria um full table scan em 70.000 linhas — ~200ms por pessoa. Com o B-tree index, cada lookup é inferior a 1ms. A diferença entre um festival fluido e um caos.
03
// O que melhoraria
Tabela dedicada de blacklist
Criaria uma tabela rfid_blacklist separada com timestamp de expiração e motivo de revogação. Mais rápida de sincronizar para os leitores offline do que filtrar toda a tabela bilhetes.
04
// O que melhoraria
Escalar para 200k pessoas
Com 200.000 participantes usaria sharding por zona do recinto (Norte/Sul/VIP em partições separadas) e um Redis cache na frente do Supabase para absorver os picos de picagem simultânea nas horas de entrada.
